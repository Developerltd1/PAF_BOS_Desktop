A)   Below Keys need to be updated for each client in App.config

    <add key="ClientID" value="1"/> <!--Unique Client ID , this key will provide by administrator--> 
    <add key="Delay" value="3"/> <!--in Seconds , Result Display Time after Finger Scan-->
    <add key="Online"  value="0"/> <!--1=Yes , 0 = NO , Use local database or LAN/WAN Database using Web service -->

B)  Incase of using local database without using webservice Connection string need to be updated accourdingly. User Name , Password , Database Name and Server.
